# Mushroom Eco Monitor  
## Система автоматической классификации грибов для задач экологического мониторинга

## Бизнес-задача

Мониторинг экосистем требует постоянного анализа данных о биоразнообразии.  
Одной из важных подзадач является идентификация грибов, так как некоторые виды являются ядовитыми и могут представлять опасность для туристов, волонтёров и исследователей.

Однако ручная классификация изображений грибов:

- требует экспертных знаний,
- занимает много времени,
- подвержена человеческим ошибкам.

**Цель проекта:** создать прототип ML-системы, которая определяет категорию гриба по фотографии:

- `edible` — условно съедобный  
- `poisonous` — потенциально ядовитый  

Такое решение может использоваться:

- для ускорения полевых исследований,
- для повышения качества мониторинга биоразнообразия,
- как вспомогательный инструмент для экологов и научных групп.

> Важно: модель является учебной, не используется для принятия пищевых решений.

## Описание ML-решения

Проект использует предобученную модель **ResNet-50 (ImageNet)**.

В системе реализовано:

- загрузка и настройка предобученной модели,
- замена выходного слоя на 2 класса (`edible/poisonous`),
- предобработка изображения (resize, normalize),
- вывод вероятностей,
- функция предсказания,
- прототип обучения на датасете грибов.
